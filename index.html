THE MAZE PROJECT BLOG

The SDL Maze project is a foundational project for the end of the trimister. The SDL game app is a C/C++ source Library tool for this project. This project is a portfolio project which is solely design by Isaac Okyere. What really inspired me to consider the maze project is the new developments in game development and techniques associated with launching a c/c++ source game programs.

The Game Control keys are listed below

keys functions
Left arrow key Use to turn left
Right arrow key Use to turn right
Up arrow key Use move up
Down arrow key Use move down
Shift key Hold to move twice as fast
t key Toggle between textured and untextured rendering
m key Toggle the full screen map on/off
f key Toggle the barrel distortion correction on/off
[ key Decrease the distance to the viewplace (increase FOV)
] key Increase the distance to the viewplace (decrease FOV)
Esc key Quit or Exit the game
